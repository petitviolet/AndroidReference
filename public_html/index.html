Eclipseについて

Eclipse
エクリプス
IDE
AndroidManifest.xml
アンドロイドマニフェスト・ドット・エックスエムエル
アプリの設定、必要な権限や、使用する画面などを記述するファイル
srcフォルダ
Source(ソース)フォルダ
.javaファイルが入っているフォルダ。(例：MainActivity.java)
resフォルダ
Resource(リソース)フォルダ
画像(drawable)やレイアウト(layout)が入っているフォルダ
res/layoutフォルダ
Layout(レイアウト)フォルダ
レイアウトファイルが入っているフォルダ(例：activity_main.xml)
res/drawable-*dpiフォルダ
Drawable(ドローワブル)フォルダ
アイコンなどの画像が入っているフォルダ(例：ic_launcher.png)
res/menuフォルダ
Menu(メニュー)フォルダ
メニュー用レイアウトファイルが入っているフォルダ。(例：activity_main.xml)
res/values*フォルダ
Values(レイアウト)フォルダ
アプリ内で使用する文章や数値を保存するファイルが入っているフォルダ。(例：strings.xml)
gen
Gen(Generated)フォルダ
自動で生成されるフォルダ（触る必要はない）
bin
Bin(Binary)フォルダ
自動で生成されるフォルダ（触る必要はない）
libs
Libs(Libraries)フォルダ
アプリで使用するライブラリを入れるフォルダ
assets
Assets(アセッツ)フォルダ
resフォルダと似ているが、使い方などが違う（あまり触らない）
proguard-project.txt
プロガードプロジェクト・ドット・テキスト
セキュリティ対策用のファイル（触る必要はない）
project.properties
プロジェクトプロパティーズ
アプリのターゲットSDKなどを記述するファイル（あまり触らない）

ショートカットキー

ステージ1

保存

Mac: command + S / Windows: Ctrl + S

変更したらまず保存！　このコマンドを使う癖をつけよう！
Eclipseでは編集するとファイル名の左横に「*」が出てくるので，これがあることに気づいたらすぐに「保存」すること！

選択

shift + 方向キー

マウスを使わなくてもドラッグと同じことができる！
コードを書いてる時によく使います。
shift + クリック で同時に複数のファイルを選択することもできる！

コピー & 貼り付け

コピー
Mac: command + C / Windows: Ctrl + C
貼り付け
Mac: command + V / Windows: Ctrl + V

右クリックからのコピー & ペーストから抜けだそう！
これを覚えるだけで作業がぐっとはやくなる！

Let's Try!

「選択」を習得！
「コピー」を習得！
「貼り付け」を習得！

元に戻す

Mac: command + Z / Windows: Ctrl + Z

作業を間違えたらまずこのコマンド！
何回も押せばどんどんもどっていくので，うまくいった状態まで戻そう！
ただし，ファイルを閉じると戻れないので注意！

Let's Try!

「元に戻す」を習得！

ステージ2

やり直す

Mac: command + Shift + Z / Windows: Ctrl + Y

「元に戻す」の逆バージョン。
戻しすぎたってときはコレ！
ただし，このコマンドはEclipse以外のアプリケーションでは違う可能性があるので注意！

検索

Mac: command + F / Windows: Ctrl + F

100行もあるコードからひとつの単語を見つけ出すのは大変！
「検索」を使えば一発で見つけられる！

全て選択

Mac: command + A / Windows: Ctrl + A

全部選択するときはドラッグや shift + 矢印 では面倒くさい。
「全て選択」を使えば一瞬！

Let's Try!

「全て選択」を習得！

ステージ3

補完

Mac: control + space* / Windows: Ctrl + Space

import

Mac: command + shift + O / Windows: Ctrl + Shift + O

フォーマット

Mac: command + shift + F / Windows: Ctrl + Shift + F
